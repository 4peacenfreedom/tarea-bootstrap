Comandos de Flex y Grid en Bootstrap

Comandos de Flexbox

  • .d-flex: Aplica display flex a un contenedor. 
    .d-flex { display: flex; }
  • .flex-row: Dirección de los elementos en fila. 
    .flex-row { flex-direction: row; }
  • .flex-column: Dirección de los elementos en columna. 
    .flex-column { flex-direction: column; }
  • .justify-content-start: Alinea los elementos al inicio del contenedor. 
    .justify-content-start { justify-content: flex-start; }
  • .justify-content-end: Alinea los elementos al final del contenedor. 
    .justify-content-end { justify-content: flex-end; }
  • .justify-content-center: Centra los elementos horizontalmente. 
    .justify-content-center { justify-content: center; }
  • .justify-content-between: Distribuye los elementos con espacio entre ellos. 
    .justify-content-between { justify-content: space-between; }
  • .align-items-start: Alinea los elementos al inicio verticalmente. 
    .align-items-start { align-items: flex-start; }
  • .align-items-end: Alinea los elementos al final verticalmente. 
    .align-items-end { align-items: flex-end; }
  • .align-items-center: Centra los elementos verticalmente. 
    .align-items-center { align-items: center; }

Comandos de Grid

  • .container: Contenedor básico para centrar el contenido y agregar padding horizontal. 
    .container { width: 100%; padding-right: 15px; padding-left: 15px; margin-right: auto; margin-left: auto; }
  • .row: Contenedor para filas de columnas. 
    .row { display: flex; flex-wrap: wrap; margin-right: -15px; margin-left: -15px; }
  • .col: Columna flexible que se ajusta según el tamaño de la pantalla. 
    .col { position: relative; width: 100%; padding-right: 15px; padding-left: 15px; }
  • .col-* Columna fija de tamaño específico (1 a 12). 
    .col-1 { flex: 0 0 8.333333%; max-width: 8.333333%; }
    .col-2 { flex: 0 0 16.666667%; max-width: 16.666667%; }
  • .col-sm-* Columna fija de tamaño específico para pantallas pequeñas (≥576px). 
    @media (min-width: 576px) { .col-sm-1 { flex: 0 0 8.333333%; max-width: 8.333333%; } }
  • .col-md-* Columna fija de tamaño específico para pantallas medianas (≥768px). 
    @media (min-width: 768px) { .col-md-1 { flex: 0 0 8.333333%; max-width: 8.333333%; } }
  • .col-lg-* Columna fija de tamaño específico para pantallas grandes (≥992px). 
    @media (min-width: 992px) { .col-lg-1 { flex: 0 0 8.333333%; max-width: 8.333333%; } }
  • .col-xl-* Columna fija de tamaño específico para pantallas extra grandes (≥1200px). 
    @media (min-width: 1200px) { .col-xl-1 { flex: 0 0 8.333333%; max-width: 8.333333%; } }
  • .g-* Espaciado entre columnas (gap) (0 a 5). 
    .g-0 { gap: 0; }
    .g-1 { gap: 0.25rem; }
  • .offset-* Desplazamiento de columnas para alinear el contenido. 
    .offset-1 { margin-left: 8.333333%; }
    .offset-2 { margin-left: 16.666667%; }
